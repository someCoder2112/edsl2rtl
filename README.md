# edsl2rtl

## Case Study ##

Encoder/decoder case study example.

### Setup ###

- Host OS: Ubuntu 22.04
- Quartus: Version 16.1 Pro Edition
- ForSyDe/Haskell: GHC 9.4.8, Stack 3.1.1, configured with GHCup
1) GHCup install:
```sh
curl --proto '=https' --tlsv1.2 -sSf https://get-ghcup.haskell.org | sh
```
2) ForSyDe install with stack:
```sh
stack update
stack upgrade
stack install forsyde-shallow
```
3) To run the model, run the following command:
```sh
stack ghci --package=forsyde-shallow srcFileName.hs
```

### Folder structure

- [model](https://github.com/someCoder2112/edsl2rtl) contains the high level modeling in ForSyDe
- [encDecMode](https://github.com/someCoder2112/edsl2rtl/tree/main/encDecMode) contains the low level mode reconfiguration implementation, generated by applying the synthesis steps presented in the paper.
- [encDecPR](https://github.com/someCoder2112/edsl2rtl/tree/main/encDecPR) contains the low level function reconfiguration implementation, generated by applying the synthesis steps presented in the paper.
